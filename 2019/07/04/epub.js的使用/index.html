npm安装

npm install epubjs

epub阅读器的开发流程

epub电子书的解析和渲染

1. 生成Book对象

this.book = new Epub(DOWNLOAD_URL)

DOWNLOAD_URL：电子书的下载地址（或者电子书在静态服务器的存贮地址）

2. 通过Book。renderTo生成Rendition对象

this.rendition = this.book.renderTo('read',{
    width: window.innerWidth,
    height: window.innerHeight,
    method: 'default'
})

‘read’：是将book选在在ID名称为read的元素下

3. 通过Rendition.display()渲染电子书

this.rendition.display()

4. epub电子书翻页功能

上一页

function prevPage() {
    if(this.rendition){
        this.rendition.prev()
    }
}

下一页

function nextPage() {
    if(this.rendition){
        this.rendition.next()
    }
}

5. epub电子书的字号设置和场景切换

需要先进行注册主题，然后再使用select调用、切换主题

themeList = [
    {
        name: 'default',
        style: {body:{color:#fff,background:#000}}
    },
    ...
    ...
]

注册主题

this.themes = this.book.themes
function registerTheme() {
    this.themeList.forEach(theme => {
        this.themes.register(theme.name,theme.style)
    })
}

设置主题

function setTheme(index) {
    this.themes.select(themeList[index].name)
    this.defaultTheme = index
}

设置字号大小

function setFontSize(fontSize) {
    this.defaultFontSize = fontSize
    if(this.themes){
        this.themes.fontSize(this.defaultFontSize = 'px')
    }
}

6. epub电子书生成目录和定位信息

book对象的钩子函数ready

book在实际解析、加载的过程中是有一个过程的，部分功能因为比较消耗性能，是需要解析后手动进行加载的，例如下面需要用到的目录（navigation）和定位（locations）对象

this.book.ready.then(() => {
    // 生成目录navigation对象
    this.navigation = this.book.navigation
    // 生成locations对象并返回，用来定位
    return this.book.locations.generate()
}).them(result => {
    // 保存locations对象
    this.locations = this.book.locations
    // 标记电子书为解析完毕状态，通过这个变量的值确定是否进行某些操作
    this.bookAvailable = true
})

电子书通过百分比进行定位

function onProgressChange(progress) {
    const percentage = progress / 100
    const location = percenter > 0 ? this.locations.cfiFromPercentage(percentage) : 0
    this.rendition.display(location)
}

7. HTML5 range控件

<input class="progress"
       type="range"
       max="100"
       min="0"
       step="1"
       @change="onProgressChange($event.target.value)"
       @input="onProgressInput($event.target.value)"
       v-bind:value="progress"
       v-bind:disabled="!bookAvailable"
       ref="progress">

说明：

@change    是监听滑块在鼠标离开时值得变化
@input    是监听滑块在被拖动时值得变化
