Markdown编辑器的基本语法

本文引自 简书：Markdown基本语法

2019/7/2 星期二 下午 2:43:17

Markdown是一种纯文本格式的标记语言。通过简单的标记语法，它可以使普通文本内容具有一定的格式。

相比WYSIWYG编辑器

优点：

  1. 因为是纯文本，所以只要支持Markdown的地方都能获得一样的边际效果，可以让作者拜托排版的困扰，专心协作；
  2. 操作简单。比如WYSIWYG编辑时标记个标题，先选中内容，再点击导航栏的标题按钮，选择几级标题。要三个步骤，而Markdown只需要在标题内容前加“#”即可。

缺点：

  1. 需要记一些语法（当然是很简单的语法，五分钟即可学会）；
  2. 有些平台不支持Markdown编辑模式（但常用的一些博客类的网站大都支持，例如简书）。

二、字体

  • 加粗

要加粗的文字左右分别使用两个**包裹起来

  • 斜体

要倾斜的文字左右分别用一个*包裹起来

  • 斜体加粗

斜体加粗的文字左右分别使用三个***包裹起来

  • 删除线

要加删除线的文字左右分别用两个~~包裹起来
